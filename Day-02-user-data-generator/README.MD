# Day 2 — User Data Message Generator (n8n Project)

## Overview

This project demonstrates how data flows inside an n8n workflow. It shows how to create user data, pass it between nodes, and generate a dynamic message using expressions.

## Workflow

Manual Trigger → Set User Data → Message Generator

## Nodes Used

* Manual Trigger
* Set Node (User Data)
* Set Node (Message Generator)

## Input Fields

Name, Age, Course, Email

## Output Example

Hello Akhil, You are 22 years old and enrolled in the Automation course. Your registered email is [akhil121@gmail.com](mailto:akhil121@gmail.com).

## Concepts Learned

* Data handling in workflows
* JSON structure
* Passing data between nodes
* Expressions for dynamic values
